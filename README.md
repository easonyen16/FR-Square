# FR-Square
前置作業
* Ffmpeg
*	Sed
*	Aria2

1. 連線VPN，執行指令
```
python get_fr_square.py
```
2. 獲得 `output.txt` 
![](https://upload.cc/i1/2021/12/08/khGPLC.png)
3. 以最新一集作範例
  * `「イノゲーム！ポーカーフェイスなのは大園？藤吉？武元？松田？」`為影片名稱
  * `24401` 為影片序號
  * `1001`、`1002`為鏡頭編號
  * `ルーズ`、`井上梨名`為鏡頭名稱
  * `24401_tracking_00`中的`00`為版本
4. 開啟`frsquare.bat'
5. 輸入影片序號，如`24401`
6. 輸入鏡頭數，如本集最大編號為`1006`，故輸入6
7. 輸入版本號，僅輸入個位數即可
